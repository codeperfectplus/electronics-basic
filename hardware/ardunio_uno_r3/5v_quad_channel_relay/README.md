Arduino Uno          5V Dual Channel Relay Module
-----------          ---------------------------
5V       --------->  VCC
GND      --------->  GND
Pin 7    --------->  IN1 (controls Relay 1)
Pin 8    --------->  IN2 (controls Relay 2)



it requires 5V

setting low on the relay pin set them on
setting high on the relay pin set them off
